## Primeira Mensagem Sugerida Após o Prompt

Após colar o prompt do agente, envie uma destas tarefas:

### Para começar a produção:
> Pesquise e me dê um guia completo de como configurar o Kling 3.0 Subject Binding via Fal.ai para manter consistência de personagem cartoon. Inclua código de exemplo em Node.js, melhores práticas de prompt engineering para cartoon style, e como criar a "Golden Reference Image" perfeita para um personagem de animação infantil brasileira.

### Para análise competitiva:
> Faça uma análise profunda dos 15 maiores canais de animação AI no YouTube (BR e global). Para cada um: ferramentas usadas, estilo visual, frequência de upload, subscribers, views médios, e estratégias de crescimento. Identifique os gaps no mercado BR que THIAGERA pode explorar.

### Para otimização de custos:
> Analise nosso pipeline completo e otimize os custos. Compare o custo de produzir 1 episódio de 10 minutos usando: (A) stack premium, (B) stack econômica, (C) stack híbrida. Inclua cada etapa do pipeline com custo individual e alternativas gratuitas/open-source.
